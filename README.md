# Instagram UI Concept Clone

A multi-page frontend UI project inspired by Instagram's design language. This project features a modular tab-based navigation system and custom-themed content discoveries, including specialized trending sections for Anime, Marvel, and Space.

## 🚀 Overview

This project is a detailed UI exploration focusing on layout, navigation flow, and thematic styling. Rather than a direct replica, it serves as a functional prototype of a social media interface, complete with authentication pages and a rich set of content discovery tabs.

## ✨ Key Features

* **Complete Auth Flow:** Dedicated, styled interfaces for both Login and Signup.
* **Modular Tab Navigation:** Seamlessly switch between Feed, Explore, Notifications, and Settings.
* **Custom Trending Modules:** Unique "Trending" tabs designed to showcase specific niche interests:
    * **Anime:** Curated content for anime enthusiasts.
    * **Marvel:** Dedicated space for cinematic and comic book updates.
    * **Space:** Exploration-themed content and visuals.
* **Responsive Design:** Consistent CSS architecture across all sub-pages and tabs.

## 📂 Project Structure

The project is organized into logic-based files for easy maintenance:


Instagram_Clone/
│
├── Core Application
│   ├── index.html             # Main Feed & Dashboard shell
│   └── instagram_clone.css              # Global styling for the main app
│
├── Authentication
│   ├── instagram_login_page.html        # User Login interface
│   ├── instagram_login_page.css         # Styling for login components
│   ├── instagram_signup_page.html       # User Registration interface
│   └── instagram_signup_page.css        # Styling for signup components
│
├── Navigation Tabs
│   ├── instagram_explore_tab.html       # Discover new content
│   ├── instagram_notification_tab.html  # User alerts and interactions
│   ├── instagram_people_tab.html        # Suggested followers and profiles
│   └── instagram_setting_tab.html       # User preferences and account settings
│
└── Thematic Trending Tabs
    ├── instagram_trending_tab.html      # General trending overview
    ├── instagram_trendinganime_tab.html # Specialized Anime content
    ├── instagram_trendingmarvel_tab.html# Specialized Marvel content
    └── instagram_trendingspace_tab.html # Specialized Space content

 ##   💻 Tech Stack
HTML5: Semantic structure for high-fidelity social media layouts.

CSS3: Advanced styling using Flexbox, Grid, and custom transitions for a smooth UI experience.

Iconography: Integrated font icons and visual assets to match the social media aesthetic.
